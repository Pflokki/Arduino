#############################
###       О проекте       ###
#############################

Данный проект зарабатывается как модуль для умного дома,
отвечающий за подсветку. Основная задача проекта - осуществлять
управление rgb-лентой.
Сигнал на смену режима микроконтроллер должен получать из внешней
среды по протоколу I2C.

#############################
###     Режим работы      ###
#############################

В проекте предусмотрено 5 основных режимов работы, а также возможность
отображения 8 фиксированных цветов. Для работы с цветом была выбрана
HSV цветовая палитра.
1 режим – отображения всего цветового спектра.
2 режим – отображение заданного диапазона из цветового спектра.
3 режим – режим рассвета.
4 режим – режим заката.
5 режим – отображение 8 фиксированных цветов один за другим.

Доступные цвета: красный, оранжевый, желтый, зеленый, синий,
фиолетовый, розовый, белый.


#############################
###       Реализация     ####
#############################

На данный момент реализовано переключение режимов через кнопку,
в дальнейшем, кнопку легко заменить на получение сигнала с управляющего
микроконтроллера по протоколу I2C. 
Для возможности обработки сигнала с управляющего микроконтроллера сразу же,
 при получении сигнала в проекте реализована «умная задержка», которая
 не тормозит работу микроконтроллера.

#############################
###      Подключение      ###
#############################

Питание ленты осуществляется через внешний блок питания на 12В,
также к этому блоку питания подключен сам микроконтроллер.
Цифровые пины 9-11 микроконтроллера соединены с базой моссфетов и
отвечают за один из цветов (красный, зеленый, синий), сток моссфетов
соединен с лентой, исток моссфета соединен с землей.
2 цифровой пин соединен с двух-пиновой кнопкой, вторая ножка кнопки
соединина с землей. 
Земля микроконтроллера соединена с землей блока питания.
